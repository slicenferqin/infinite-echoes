# ep01 对话谎言模型（V2）

目的：把“口供层级”做成可拆穿结构，支持信任、威胁、证据三条动态路径。

## 一、统一口供层
- 低信任层（防御）：
  - 关键词：不知道、没看见、别问我。
  - 产出：情绪、立场，不给可执行线索。
- 中信任层（松口）：
  - 关键词：我听说、也许、你别外传。
  - 产出：方向性 lead（可引导搜查）。
- 高信任层（真相）：
  - 关键词：我亲眼、我愿作证、就是这样。
  - 产出：可验证证词、关键线索或权限。

## 二、拆谎触发方式
- 方式A：共情推进（低风险）
  - 适用：诺拉、玛莎、米拉、汉克。
  - 效果：稳步提升信任，减少敌意。
- 方式B：施压推进（高风险）
  - 适用：雷恩、艾德蒙（短时有效）。
  - 风险：威胁值升高，可能触发冷却/失联。
- 方式C：出示证据（精准突破）
  - 适用：全体关键 NPC。
  - 效果：直接击穿一层口供，触发“态度跃迁”。

## 三、NPC 主题谎言卡（精简）
- 诺拉
  - 主题：父亲夜间行为。
  - 真相触发：谈“护人代价”+信任阈值。
- 汉克
  - 主题：沉默原因。
  - 真相触发：先证明你知道其风险，再给“护女退路”。
- 艾德蒙
  - 主题：雷恩口供与书房文书。
  - 真相触发：证据足够 + 体面退场预期。
- 玛莎
  - 主题：目击与赌桌威胁。
  - 真相触发：安全承诺 + 分步提问。
- 雷恩
  - 主题：当晚行踪、血衣去向。
  - 真相触发：证据拆矛盾（一次压一条）。
- 加雷斯
  - 主题：矿工是否愿作证、脚印判断。
  - 真相触发：先谈工友利益再谈细节。
- 米拉
  - 主题：伤情判断、夜间工况。
  - 真相触发：按时间线提问，避免情绪绑架。

## 四、谈崩恢复机制
- 状态定义：
  - `open`（开放）→ `guarded`（防备）→ `hostile`（敌对）。
- 恢复规则：
  - `hostile` 需至少 1 个时段冷却，或“共情 2 次 / 证据修复 1 次”恢复。
  - 恢复后默认回到 `guarded`，不会立刻完全开放。
- 代价：
  - 失去当前时段推进机会，部分事件可能错过（如 `B5` 销毁窗口）。

## 五、与搜查门槛联动
- 关键搜查点（B4/B5/C2/C5）默认需要 lead。
- lead 的主要来源是对话层跃迁，而非地图点点击。
- 结果：纯搜查只会获得表面叙事，难达非 BE 结局。
