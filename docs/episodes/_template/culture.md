# 副本文化规范模板

## 1. 价值核心
- 列出 2-4 个价值锚点（如：亲情与责任、人情与公义、乡里秩序）。
- 说明冲突时的优先级（例：先情后理，最终落法）。

## 2. 情绪表达
- 表达风格：`克制含蓄` 或 `直给浓烈`。
- 关键桥段必须通过“行动 + 对话”双轨呈现，不只写独白。

## 3. 禁用表达
- 禁止网络梗、现代职场黑话、欧美法庭剧口吻。
- 禁止无因由的个人英雄主义台词。

## 4. 关系秩序
- 明确角色关系位阶（长幼、恩义、抚养、同乡、债务）。
- 每个关键行为要能用关系位置解释“为什么这么做”。

## 5. 评审清单
- 是否有至少 2 个“情/理/法”冲突节点？
- 是否存在“错的人不一定坏，守法也可能伤人”的复杂性？
- 结局遗憾是否来自现实代价而非生硬反转？
