# EP02 对话谎言分层（V2）

## 设计原则
- 低信任：只给表层事实或防御性说法。
- 中信任：给方向，不给闭环。
- 高信任：给可执行的证据入口（lead 或关键口供）。

## 关键话题映射
- `lead_guardianship_chain`
  - 主要来源：阿哲 / 莉娜 / 萨拉
  - 话题关键词：监护、债主、家产、代管
- `lead_private_route`
  - 主要来源：魏德 / 萨拉 / 阿哲
  - 话题关键词：私驿、后港、小船、绕登记
- `lead_warehouse_key`
  - 主要来源：魏德 / 莉娜 / 萨拉
  - 话题关键词：铁箱、钥匙、暗号、仓库
- `lead_notary_fee_code`
  - 主要来源：诺埃尔 / 马修 / 魏德
  - 话题关键词：费码、手续费、暗账、缩写
- `lead_old_case_index`
  - 主要来源：诺埃尔 / 马修 / 萨拉
  - 话题关键词：旧案、索引、案号、七起

## 谈崩恢复钩子
- 共通恢复：承认刚才施压过重 + 给出具体保护方案 + 回到可复核问题。
- 特殊恢复：
  - 魏德：先谈家属保护，再谈交易细节。
  - 阿哲：先谈生存安排，再谈证词。
  - 诺埃尔：先给合法调档理由，再问编号链。
