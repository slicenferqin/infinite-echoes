# 副本 02：迟到的回执（The Delayed Receipt）

## 基本信息
- 难度：★★☆☆☆
- 情绪基调：温情 + 遗憾（轻刀）
- 外壳场景：边境山港小镇（西式地理皮相）
- 内核冲突：中式“情、理、法”排序冲突
- 时间机制：2 天实时审理窗口（1 小时/天，共约 120 分钟）
- 路线：HE / TE / BE / SE

## 一句话钩子
“她违了规矩，救了一个家；却让另一个家再也等不到那封信。”

## 案件背景
灰石港是一座靠码头和驿站运转的小镇。驿站掌柜莉娜被指控私扣抚恤回执，受害人乔安在等待中病故，留下未成年儿子阿哲。

司法官马修宣布：从现在起两天内完成初审。舆论却只剩一句话——“人都死了，还查什么？”

玩家的表层任务是查“谁扣了信”，深层任务是判断：
- 违规救人，是否就是恶？
- 合法定罪，是否就等于公道？

## 四层真相
1. **表层**：莉娜确实延迟了回执。
2. **中层**：延迟是为了切断债主对阿哲监护权的围猎。
3. **深层**：公证所与代投网络长期利用“时间差”牟利。
4. **隐层**：同编号双回执出现，暗示记录系统被人为改写。
