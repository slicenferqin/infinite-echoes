# 副本 02 路线与结局（V6.2）

## HE：灯未熄（温情遗憾）
- 条件核心：证明莉娜并非谋私 + 保护阿哲监护权 + 揭开代投链。
- 社交硬条件：必须接触 `lina/azhe/matthew`，且 `lead_` 数量 >= 3。
- 结果：莉娜免重罪，阿哲保住家产；但乔安已无法归来。

## TE：秤与债
- 条件核心：完整拉出 C1/C2/C4 结构证据，坐实公证-代投勾连。
- 社交硬条件：必须接触 `weide/noel/matthew`，且 `lead_` 数量 >= 4。
- 结果：制度纠偏启动，关键证人背负“背乡”代价。

## SE：同编号之夜
- 条件核心：抓住 C1 并明确“记录层被改写”的推断。
- 社交硬条件：必须接触 `weide/noel`，且 `lead_` 数量 >= 3。
- 结果：案件重审启动，元叙事异常显现。

## BE：按章定罪
- 条件核心：只完成表层程序链，未撬动社交证据网络。
- 结果：程序闭环成立，但结构性牟利者继续隐身。

## 非 BE 资格门槛（全局）
- 唯一接触 NPC 数量不足，或未触发最低 lead 链时，系统会强制滑向 BE。
- 该机制用于放大“社交调查”而非“地图速通”。
