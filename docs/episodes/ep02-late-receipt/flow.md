# 副本 02 流程（V6.2）

## 进入阶段
- 玩家在灰石港公告栏前接到案件。
- 马修明确：两天实时窗口结束即定性，过时不候。

## 探索阶段（Day1 / Day2）
- 可调查区域：驿站、码头账房、公证所、贫民巷、旧仓库、港口互助摊周边。
- 推进主逻辑：
  1. 先通过 NPC 对话拿 `lead_` 线索方向。
  2. 再去对应地点搜查拿核心 clue。
  3. 最后回到制度链完成可复核提交。

## 时段事件（强约束）
- Day1 傍晚：魏德受压，口风收缩。
- Day2 上午：若未接触魏德，进入改口态。
- Day2 傍晚：公证账页焚毁，关键物证窗口关闭。

## 提交阶段
- 必答三问：
  1. 延迟责任是谁、怎么发生？
  2. 动机属于“谋利”还是“护人”？
  3. 程序、公义、人情的裁断顺序是什么？

## 结算阶段
- 先落结局，再给伦理评估（情/理/法取舍）。
- SE 时触发“同编号回执”异常指向。
- 结算后进入阿寂主动对话过场。
