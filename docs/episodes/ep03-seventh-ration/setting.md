# 副本 03：第七天配给（The Seventh-Day Ration）

## 基本信息
- 难度：★★★☆☆
- 主线定位：中章升级（承接 EP02）
- 情绪基调：压抑克制 + 情感爆点
- 时间机制：7 天实时制（30 分钟/天，约 3.5 小时）
- 外壳：高科技配给城
- 内核：中式“养育责任 vs 制度正当性”

## 一句话钩子
“系统判她不再具备抚养资格，但她是最后一个记得孩子生日的人。”

## 世界线连接
- EP02 的“记录被改写”异响触发回响层切换。
- 玩家进入第七层配给城，发现 AI 托管秩序与高维入侵防线绑定。
- 阿寂从旁白升级为可对话关键 NPC。

## 四层真相
1. 表层：配给失踪与监护降级让母子濒临分离。  
2. 中层：规则被人借用，配给路由与信用条款被操控。  
3. 深层：AI 主动压缩真相以维持防线稳定。  
4. 隐层：记录改写是防线策略副产物，也成了权力滥用入口。  

## 设计目标
- 玩家能同时理解“AI 非纯恶、人类也非纯善”。
- 强化社交调查：不走 NPC 关系链，关键搜查不开门。
- 结局保持双正确双代价，而非单一价值灌输。
