# 副本 03 流程（V1）

## 开场
- 回响层切换，进入第七层配给城。
- 玩家接到余岚母子监护危机求助。

## 中段推进
1. 通过 NPC 对话解锁社交 lead。  
2. 再进入对应地点搜查关键 C 级证据。  
3. 把“家庭线、制度线、防线线”三条链拼接成可复核结论。  

## 七天关键事件
- Day2 傍晚：余岚抚养资格冻结
- Day3 夜间：灰市口风收紧
- Day4 正午：防线伤亡明细被裁剪
- Day5 傍晚：核心后门封补
- Day6 夜间：配给路由重定向
- Day7 傍晚：审理自动闭合，强制结算

## 结算
- 先判路线，再给价值总结。
- 总结必须体现“双正确双代价”，禁止脸谱化结论。
