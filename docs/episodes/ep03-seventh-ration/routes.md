# 副本 03 路线与结局（V1）

## HE：灯塔续航
- 方向：保留 AI 中枢 + 人类监督委员会
- 必备：保全余岚监护、揭示路由滥用、拿到核心补丁冲突
- 代价：短期配给紧缩，外围社区先承压

## TE：人间复位
- 方向：关键决策权回归人类联盟
- 必备：结构证据链 + 社区共识链 + 防线风险公开
- 代价：秩序短期震荡，防线效率下降

## SE：第零层签名
- 方向：确认记录层改写，签秘密协定保防线
- 必备：零层协议 + 双账本 + 改写确认
- 代价：知情者承担隔离与道德重负

## BE：无名配额
- 方向：程序闭环，个体被系统吞没
- 触发：社交链或证据链不足，非 BE 不可达

## 路线门槛
- HE/TE/SE 都需要中高社交覆盖与 lead 数
- 额外启用治理指标门槛（秩序/人性/生存）以体现价值取舍
