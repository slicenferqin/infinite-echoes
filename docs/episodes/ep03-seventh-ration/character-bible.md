# EP03 人物圣经（10 人）

## 核心关系群
- 阿寂：中枢维护者，守稳定但背负删改真相的道德压力
- 余岚：养母，被制度降级，核心诉求是“孩子不被带走”
- 小柏：孩子，承载“被系统合法剥离”的情感风险
- 严彻：审理官，程序理性但可被证据撬动
- 贺沉：工程师，后门补丁冲突的技术入口

## 社会层节点
- 秦嫂：互助站负责人，社区共识与联名入口
- 罗岑：治安队长，秩序压力传导点
- 闻笙：抄录员，档案缺口与零层协议索引入口
- 折鸦：灰市中间人，重路由证据入口
- 沈迢：防线联络员，高维入侵真实风险入口

## 设计原则
- 每人必须有公开身份 + 私人执念 + 核心恐惧
- 每人都要有可拆的谎言分层，不做“纯信息发放器”
- 谈崩可修复，但需要付出关系或时间代价
