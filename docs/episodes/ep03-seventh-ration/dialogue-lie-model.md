# EP03 对话谎言分层（V1）

## 分层原则
- 低信任：保护性叙述 / 含混说法
- 中信任：给方向，不给闭环
- 高信任：给可复核入口或可执行证据线

## 关键 lead 对应
- `lead_care_record_gap`：闻笙 / 秦嫂 / 阿寂
- `lead_credit_rule`：严彻 / 闻笙 / 贺沉
- `lead_supply_reroute`：折鸦 / 秦嫂 / 小柏
- `lead_breach_suppressed`：沈迢 / 贺沉 / 阿寂
- `lead_core_backdoor`：贺沉 / 阿寂 / 闻笙
- `lead_zero_layer_protocol`：阿寂 / 沈迢 / 闻笙

## 谈崩恢复规则
- 先承认对方承担的现实代价，再继续追问
- 不允许连续两轮高压逼供同一 NPC
- 关键 NPC（余岚/沈迢/折鸦）必须有替代提问路径，避免死锁
