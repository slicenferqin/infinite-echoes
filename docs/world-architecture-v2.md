# 无尽回响（Infinite Echoes）— 世界架构 v2

> 本文档用于替代早期“副本 + 元叙事草案”的分散描述，统一当前产品方向、世界观结构与后续实现边界。  
> 目标不是写一份文学设定集，而是给后续副本设计、系统设计、Agent 设计提供同一套约束。

> 对应执行路线图：`/Users/slicenfer/Development/projects/self/infinite-echoes/app/docs/roadmap-v2-execution.md`

---

## 一、产品定位（正式定案）

**一句话定义：**  
《无尽回响》不是“很多 AI NPC 的文字游戏”，而是一个**以文字为主要媒介的持久世界叙事引擎**。

玩家体验到的是：

- 每次进入一个**独立但真实的回响层世界**
- 在其中与有记忆、有关系、有立场、有谎言逻辑的人互动
- 在制度压力与情感冲突中完成调查、判断与提交
- 再从这些副本世界回到一个持续存在的元世界
- 逐步意识到：这些副本不是随机故事，而是被某种更大结构设计出来的“情绪生产装置”

因此，本项目的正式结构不是：

- 单一表层世界的长篇连续剧  
也不是
- 完全割裂的 anthology 副本合集

而是：

**表层多世界，深层同一元世界。**

更准确地说，它是：

**伪 anthology，真主线宇宙。**

---

## 二、为什么不做“所有副本同一个表层世界”

这是 v2 的第一条关键决策。

### 2.1 不统一表层世界的原因

如果所有副本都发生在同一个地理与社会现实层里，文字游戏会很快遇到三个问题：

1. **审美疲劳**  
   玩家看到的只是换案件，不是换世界。文字产品尤其依赖世界差异来制造新鲜感。

2. **题材收缩**  
   你很难同时合理容纳乡土伦理、边港制度、AI 治理社会、宗教秩序、战时社会、学院政治等题材。

3. **元叙事被削弱**  
   如果一切都在一个表层世界里，玩家会更容易把它理解为普通连续剧情，而不是“被组织起来的世界切片”。

### 2.2 保留多世界的收益

保留表层多世界，反而更适合《无尽回响》：

- 每个副本都能拥有**强烈的时代感、制度感与情感母题差异**
- 每个副本都能承载一种不同的“人性/秩序实验”
- 玩家会天然感受到“这些世界为什么会被挑中”
- 元世界的存在感会更强，因为它成为所有差异世界背后的唯一连续物

所以 v2 明确规定：

> **每个副本首先必须作为“独立世界切片”成立；其后才允许通过元世界进行关联。**

## 三、三层世界结构

v2 将世界正式拆为三层。

## 3.1 第一层：元世界层（回响之间）

这是玩家真正“持续生活”的地方。

元世界的作用不是提供大地图探索，而是承担：

- 跨副本连续性
- 常驻 NPC 关系发展
- 档案与记录管理
- 副本选择/开启的解释
- 回响柱、裂隙门、记忆商店、档案馆等长期结构
- 主线真相推进

元世界负责回答：

- 玩家为什么会进入副本
- 为什么每个副本都像一场被精心编排的试炼
- 谁在看、谁在记录、谁在干预
- 情绪为什么会成为资源

元世界是**连续的“家”**，副本是不断进入的“切片世界”。

## 3.2 第二层：回响层（副本世界）

每个副本都是一个**独立回响层**。

回响层的定义不是“平行宇宙”，也不是“纯幻境”，而是：

> 被回响柱或其背后的维护系统调用、构造、稳定并运行的一段完整世界切片。

一个回响层必须具备：

- 自洽的社会规则
- 明确的制度压力
- 可追踪的人物关系网
- 局部历史与传闻结构
- 可被玩家改变但不可完全控制的事件链

玩家进入时看到的是“一个案子”，但系统真正部署的是“一段世界结构”。

## 3.3 第三层：收割/维护层（隐层组织）

这是 v2 新增的明确层。

早期文档只写了“回响柱在收割情感”，但没有把“谁在维持系统运行”说清楚。  
v2 规定：在回响柱之下，存在一个**维护与抄录结构**，暂定称为：

**回响维护层**

它不是传统意义上的“反派组织”，而是一个混合结构：

- 一部分像规则执行系统
- 一部分像档案与记录维护者
- 一部分像制度工程师
- 一部分像被困其中、逐渐与系统绑定的人

阿寂的存在，就是连接元世界层与维护层的关键锚点。

这层的职责包括：

- 选择或构造回响层
- 维护记录一致性
- 在必要时改写公开版本与隐藏版本
- 对情绪产出进行归档、调度、转化
- 在系统稳定与个体公义之间不断做取舍

这样一来，玩家面对的就不是“神秘石柱自动作恶”，而是一整套更复杂的结构：

- 柱子提供规则
- 系统提供维持
- 人与半人化角色在其中承担执行与犹疑

这能显著提高元叙事厚度。

## 四、七个核心承载维度

你前面提到的七个关键词：**关系、记忆、立场、谎言、制度、历史后果、模拟世界**，v2 将它们正式定义为系统骨架。

## 4.1 关系（Relationship）

NPC 不是线索发放器，而是处于关系网络中的人。

每个 NPC 的行为必须首先由“关系位置”解释，而不是由“剧情功能”解释。

关系要素至少包括：

- 亲属/抚养/师徒/同乡/同事/上下级
- 依赖关系
- 恩义关系
- 共同秘密
- 面子与体面压力
- 阵营归属

设计要求：

- 每个关键 NPC 至少连接 2 条以上关系边
- 每条核心真相必须经过至少 1 条关系链，而不是只靠地点搜查
- 玩家若不理解关系网络，就难以理解“为什么这个人现在说真话/假话”

## 4.2 记忆（Memory）

记忆必须分层，而不是只有“对话历史”。

v2 规定记忆分四层：

1. **个体记忆**  
   NPC 自己经历过什么、见过什么、害怕什么

2. **关系记忆**  
   某人和某人之间发生过什么，是否亏欠、误会、背叛、照料

3. **制度记忆**  
   这个社会过去如何处理类似事件，有哪些旧例、旧伤、旧规矩

4. **历史记忆**  
   这个世界切片过去发生过什么大事，影响到如今的口风与秩序

玩家跨副本还要有第五层：

5. **元世界记忆**  
   玩家在回响之间见过谁、档案里读到什么、哪些异常已经出现过

## 4.3 立场（Stance）

立场不等于善恶。

v2 规定所有关键角色都必须拥有可解释的立场组合，而不是单轴动机。  
建议至少按三轴定义：

- **情感立场**：护谁、怕谁、亏欠谁
- **制度立场**：守规则、钻规则、改规则、毁规则
- **生存立场**：求稳、冒险、自保、牺牲他人

这样角色之间的冲突才会真实。

例如一个人可能：

- 情感上是好人
- 制度上是冷的
- 生存上又是懦弱的

这才会形成可玩的灰度。

## 4.4 谎言（Lie）

谎言不是“为了制造悬疑”，而是角色保护自身立场的行为。

v2 规定所有关键谎言都必须回答四个问题：

1. 他在隐瞒什么  
2. 他为什么现在不能说  
3. 他对谁可以说到哪一步  
4. 什么条件下会松口

因此谎言必须是分层的：

- 低信任：完全否认或转移
- 中信任：承认一半但改写责任
- 高信任：给出真相，但仍保留某些代价判断

## 4.5 制度（Institution）

制度不是背景板，而是推动剧情的“看不见的手”。

每个副本都必须有清晰的制度压力源，例如：

- 村庄舆论与乡里秩序
- 港口票据、公证与回执系统
- AI 信用治理与配给规则
- 教团审判与赎罪法
- 战时配给与军令链

制度的存在意义是：

- 给 NPC 选择增加现实代价
- 让“守法也会伤人”成立
- 让玩家理解：错的人未必是坏人，坏结果也不总来自恶意

## 4.6 历史后果（Historical Consequence）

副本不能打完就像没发生过。

但后果也不意味着所有副本共享同一个表层世界。

v2 规定后果分三种：

1. **副本内后果**  
   同一局里，事件窗口关闭、证据销毁、NPC 改口、舆论翻转

2. **元世界后果**  
   通关后改变玩家与常驻 NPC 的关系、解锁档案、触发新异常、影响后续对话

3. **主线认知后果**  
   玩家对回响柱、维护层、复活规则、情绪燃料机制的理解发生变化

只有第三类后果，才是真正跨副本累积的核心。

## 4.7 模拟世界（Simulation World）

模拟世界不等于“几百个 agent 同时跑”。

对当前产品阶段，模拟世界的定义应该更克制：

> 让玩家感受到：即使自己不动作，世界中的角色、关系、舆论与制度也会继续变化。

也就是说，模拟感来自：

- 时间推进
- 口风变化
- 资源窗口关闭
- 情绪状态转移
- 关系链发酵
- 记录被修订

不是来自大规模 agent 数量，而是来自**状态持续变化的可信度**。

## 五、文字游戏为什么足以承载这套架构

v2 明确判断：**文字不是权宜之计，而是当前最适合《无尽回响》的主承载媒介。**

因为这个项目真正想表达的，不是物理世界的空间操作，而是：

- 谁在护谁
- 谁在怕什么
- 谁在说谎
- 为什么制度把人逼成这样
- 哪段历史还在继续伤人
- 你做的判断为什么会改变世界结构

这些内容天然适合文字。

文字的优势在于：

- 成本低，允许快速扩展世界切片
- 可容纳复杂心理与关系细节
- 可通过叙事浓度强化情感张力
- 容易支持多题材跳转，而不需要统一视觉资产

文字的短板在于视觉 spectacle 和空间反馈。  
所以 v2 不追求“文字版开放世界”，而追求：

**高密度社会模拟 + 可交互叙事 + 可积累的世界认知。**

## 六、副本设计模板（v2 统一规范）

从 v2 开始，每个副本都必须回答下面 8 个问题。

### 6.1 情感母题

这个副本要打中玩家什么情绪核心？

例如：

- 父女 / 舆论
- 人情 / 公义
- 养育 / 制度
- 信仰 / 求生
- 忠诚 / 背叛

### 6.2 制度压力

这个世界里最压人的规则是什么？

### 6.3 关系主链

哪几条关系是推动真相的主链？

### 6.4 谎言主链

哪些人分别在隐瞒什么？为什么？

### 6.5 历史阴影

这个社会里过去发生了什么，今天仍在影响案件？

### 6.6 不可逆窗口

哪些时段过去后，口供、物证、舆论或权限会发生真实变化？

### 6.7 元叙事钩子

这个副本要为回响之间的主线补哪一块认知？

### 6.8 观测副产物

这个副本打完后，哪些内容可以被整理成“可观看内容”？

例如：

- 世界日报
- 人物日志
- 冲突回放
- 档案摘要
- 某个 NPC 的未寄信件

这一条很重要。因为它决定了项目未来是否能从“可玩”扩展到“可看”。

## 七、跨副本连续性的正确承载方式

跨副本连续性不应该由“同一地理世界”承担，而应该由以下四样东西承担：

## 7.1 常驻角色连续性

阿寂、林鹿、老赵、兜帽人等，是玩家真正长期积累情感和认知的对象。

## 7.2 档案连续性

玩家每通关一个副本，不只是拿奖励，而是在档案馆里多出一块可回看的历史碎片。

## 7.3 异常连续性

回响柱共鸣、记录改写、重复编号、隐藏版本、沉默的第七柱等异常，需要在多个副本中重复出现并逐渐放大。

## 7.4 认知连续性

玩家真正带走的“成长”不是数值，而是：

- 我现在知道这系统不是中立的
- 我开始怀疑复活规则
- 我发现有公开版本与隐藏版本
- 我意识到有些角色不是纯恶，也不是纯善

这才是《无尽回响》跨副本的连续感来源。

## 八、与当前实现的对齐关系

目前代码实际上已经部分走到了 v2 方向上，只是文档没有同步。

### 已经成立的部分

- `身份池`：玩家不是抽象提问者，而是在世界中有社会位置
- `社交 lead`：关键线索前移到关系与对话
- `socialEligibility`：非 BE 路线要求建立真实互动链
- `realtime_day`：世界会因为时间推进而发生状态变化
- `runtime hooks`：支持不可逆窗口、舆论变化、证据关闭
- `metaNarrativeHooks`：允许副本结算后继续推动元世界主线

### 尚未完成的部分

- 元世界常驻 NPC 的正式数据结构
- 跨副本玩家认知/关系状态的持久记录
- 档案馆/记忆商店/裂隙门的产品化 UI
- 副本后果如何反哺后续副本的明确规则
- “世界副产物内容层”（日报、日志、异闻）生成

## 九、后续系统扩展建议

下面这些不是立即上线需求，而是 v2 之后的正确扩展方向。

## 9.1 元世界状态层

建议后续增加：

- `MetaWorldState`
- `PersistentNpcState`
- `ArchiveEntry`
- `AnomalyTrack`
- `PlayerCognitionFlags`

它们负责记录：

- 玩家与常驻角色关系
- 已知异常
- 已解锁主线认知层级
- 哪些档案被读取过
- 哪些回响柱出现过异常反应

## 9.2 制度 / 阵营状态层

建议为后续副本引入：

- `FactionState`
- `InstitutionPressure`
- `PublicOpinionTrack`

这会让“制度”从副本文案升级为真正可推演的结构。

## 9.3 世界副产物生成层

建议在副本结算后自动生成：

- 一份世界日报
- 一份档案摘要
- 一份角色碎片文本
- 一份异常记录

这一步会成为未来“可观看内容层”的雏形。

## 十、明确不做的事

为了防止方向再次发散，v2 明确以下边界。

### 10.1 现阶段不做

- 单副本内数百 NPC 的大规模社会模拟
- 传统开放世界地图
- 高频动作战斗玩法
- 复杂数值养成
- 完整经济沙盒
- 以视觉演出为主的沉浸体验

### 10.2 现阶段重点做

- 高密度关系网
- 可信的记忆与谎言分层
- 制度压力与情感冲突
- 跨副本元世界推进
- 副本后的认知与异常积累

## 十一、当前阶段的产品路线

v2 推荐路线如下：

### Phase 1：副本引擎成立

目标：证明“AI 角色 + 社交推进 + 多结局”成立。

### Phase 2：元世界成立

目标：让玩家感受到自己不是在刷关，而是在回响之间长期生活。

### Phase 3：后果成立

目标：让副本不再是一次性故事，而是主线认知的长期积累。

### Phase 4：可看内容成立

目标：让世界不仅能玩，还能产出值得观看与传播的内容。

### Phase 5：模拟世界成立

目标：让玩家感受到即便不操作，世界也会继续变化。

---

## 十二、最终定案

《无尽回响》的正确方向不是：

- 做成“很多 NPC 的聊天游戏”
- 做成“同一个世界换案子”
- 做成“临时组队做任务的 agent workflow”

而是：

> 用文字构造一个由多个回响层组成的主线宇宙。  
> 每个副本是一个独立成立的制度与情感世界切片；  
> 玩家在其中通过关系、记忆、立场、谎言和制度压力推进真相；  
> 再把这些经历带回回响之间，逐步看见一个更大、更冷、更真实的结构。

这才是《无尽回响》的产品核心。

## 十三、现有副本的定位校准

为了让后续设计不再漂移，当前三个副本在 v2 中的定位明确如下：

### EP01《沉默的铁匠铺》

- **表层世界类型：** 乡土村庄回响层
- **制度压力：** 乡评、村庄秩序、次日审判
- **情感母题：** 父女 / 舆论
- **核心作用：** 教玩家理解“关系链比地点线更重要”
- **元叙事任务：** 让玩家第一次意识到回响柱会对情绪波动产生异常反应

### EP02《迟到的回执》

- **表层世界类型：** 边境山港回响层
- **制度压力：** 驿站、公证、回执与代投网络
- **情感母题：** 人情 / 公义
- **核心作用：** 让玩家看见“守规矩也可能伤人，违规则未必出于私恶”
- **元叙事任务：** 让阿寂第一次主动松口，暗示“结局可能被预写”

### EP03《第七天配给》

- **表层世界类型：** 高治理配给城回响层
- **制度压力：** 配给链、监护资格、信用规则、防线稳定
- **情感母题：** 养育 / 制度
- **核心作用：** 把“系统并非纯恶、人类也非纯善”正式引入主线
- **元叙事任务：** 让玩家第一次接触公开记录与策略记录的分裂

### 对 EP04 之后的统一要求

从 EP04 开始，新的副本不再只回答“这次案件是什么”，还必须回答：

- 这次要让玩家看见哪一种新制度？
- 这次要把哪种关系伤口推到台前？
- 这次要揭示回响维护层的哪一段新功能或新裂痕？
- 这次副本结束后，元世界里什么会被永久改变？
