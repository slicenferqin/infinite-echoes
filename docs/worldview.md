# 无尽回响（Infinite Echoes）— 世界观设定

> 注：本文件为早期 `v0.1` 草案，已与当前实现存在明显漂移。  
> 当前应优先参考：`/Users/slicenfer/Development/projects/self/infinite-echoes/app/docs/world-architecture-v2.md`

## 一、核心概念

**类型：** AI 驱动的无限流文字解谜游戏
**核心体验：** 玩家通过与 AI NPC 自由对话，收集线索、识别谎言、推导真相，在多路线分支中做出抉择。

---

## 二、元世界：回响之间（The Echo Between）

### 2.1 起源

世界的尽头不是虚无，而是一片灰白色的静默空间——**回响之间**。

这里是死亡与重生之间的裂隙。那些在现实世界中因"不该死的意外"而死去的灵魂，会被这片空间捕获。坠楼、车祸、医疗事故、雷击……这些灵魂本不该在此刻消亡，他们的"命运余量"尚未耗尽，于是被回响之间拦截。

管理这片空间的，不是神，而是**七座无名石碑**——被称为"回响柱"。没有人知道它们是谁建造的，也没有人见过它们的主人。它们只是存在着，沉默地运转着某种古老的规则。

### 2.2 空间结构

回响之间是一个有限但边界模糊的空间，视觉上呈现为：

- **中央广场：** 七座回响柱环绕排列，地面是灰白色的石板，没有天空，头顶是无尽的灰雾
- **居住区：** 广场周围自然生长出简陋的建筑，是被困灵魂们自发搭建的栖所
- **裂隙门：** 广场边缘不定期出现的光裂——通往副本的入口
- **记忆商店：** 一个由最早的被困者建立的交易场所，用"回响点"交换物资和信息
- **档案馆：** 记录所有已完成副本的真相档案，但只有通关后才能查阅对应副本

### 2.3 规则

回响柱以低沉的共鸣传达规则，所有被困灵魂都能"听懂"：

1. **你已死亡，但命运未尽。** 完成足够的回响试炼，你的命运余量将被填满，届时你可以选择：重返人间（复活），或永远留在回响之间。
2. **每隔一定周期，裂隙门将开启。** 你必须进入。拒绝进入者，将被回响柱抹除——不是死亡，是彻底的消失，连灵魂都不剩。
3. **副本中的死亡不会让你消失，** 但会扣除命运余量。余量归零，等同于被抹除。
4. **你在副本中的一切行为都会被记录。** 回响柱在看着。
5. **不得将回响之间的信息带入副本世界。** 副本中的你，只是一个"恰好出现在那个世界的旅人"。

## 三、玩家设定

### 3.1 身份

玩家是一个因意外死亡的现代人。游戏开始时，玩家需要设定：

- **死因：** 从几个选项中选择（或自定义），这会影响某些副本中的隐藏对话
- **生前职业：** 赋予玩家一个被动特质，例如：
  - 记者 → "敏锐直觉"：每个副本额外获得 1 条免费线索提示
  - 医生 → "察言观色"：能感知 NPC 的情绪波动描述更细腻
  - 程序员 → "逻辑链"：提交推理时容错率略高
  - 教师 → "循循善诱"：NPC 信任度提升更快
- **名字：** 自定义

### 3.2 核心属性

| 属性 | 说明 |
|------|------|
| **命运余量** | 生命值的概念。初始 100，副本失败扣除，归零则永久死亡。通关可恢复。 |
| **回响点** | 通用货币。副本结算获得，可在记忆商店消费。 |
| **洞察力** | 累积属性。随着通关副本数增加而提升，解锁更高级的感知描述。 |
| **声望** | 在回响之间的社交资本。影响其他被困者对你的态度和可交换的信息。 |

### 3.3 成长机制

玩家不会变强——没有等级、没有技能树、没有装备。成长体现在：

- **信息积累：** 通关副本后解锁的档案，可能包含对未来副本有用的知识
- **关系网络：** 回响之间的其他被困者（常驻 NPC），随着交互加深会提供帮助
- **玩家自身的推理能力提升：** 这是真正的成长——你变聪明了，不是角色变强了

## 四、副本系统

### 4.1 副本结构

每个副本是一个独立的微型世界，有自己的时代背景、地理环境和人物关系。

**副本要素：**

```
副本名称：「沉默的铁匠铺」
时代背景：中世纪欧洲风格小村庄
难度等级：★☆☆☆☆（入门）
回合上限：30 回合
地点数量：6 个
NPC 数量：5 个
路线数量：3 条（HE / TE / BE）
隐藏要素：1 条隐藏线索链，1 个隐藏结局（SE）
```

### 4.2 副本流程

```
【进入阶段】
  裂隙门开启 → 玩家被传送 → 以"旅人"身份出现在副本世界
  ↓
【任务简报】
  GM 以叙事方式交代背景（不是弹窗，是融入剧情的信息）
  例：你走进村庄时，一个慌张的女孩拦住你——
     "求求你，帮帮我父亲，他被指控杀了人，但他没有！明天就要行刑了！"
  ↓
【自由探索阶段】（核心玩法，消耗回合）
  - 移动到不同地点
  - 与 NPC 对话（自由输入）
  - 观察环境（GM 描述场景细节）
  - 特殊行为（翻找、偷听、跟踪等，需要文字描述意图）
  ↓
【推理提交】
  玩家随时可以向"审判者"（副本内特定 NPC 或机制）提交推理
  用自然语言描述：谁做了什么、为什么、证据是什么
  ↓
【结算阶段】
  GM Agent 评估推理完成度 → 判定路线 → 播放结局叙事 → 评分
```

### 4.3 回合机制

- 每个副本有固定回合上限（入门 30，普通 25，困难 20）
- **消耗回合的行为：** 与 NPC 进行一次完整对话、移动到新地点、执行特殊行为
- **不消耗回合的行为：** 查看已收集的线索、查看地图、整理笔记
- 回合耗尽时强制进入结算，根据当前收集的信息判定结局（通常是 BE）

### 4.4 难度分级

| 等级 | 回合 | NPC | 特点 |
|------|------|-----|------|
| ★☆☆☆☆ 入门 | 30 | 4-5 | NPC 较坦诚，线索明显，单一案件 |
| ★★☆☆☆ 普通 | 25 | 5-7 | 有 NPC 会说谎，需要交叉验证 |
| ★★★☆☆ 困难 | 22 | 6-8 | 多条案件线交织，NPC 有隐藏身份 |
| ★★★★☆ 噩梦 | 20 | 7-10 | NPC 主动误导，证据可能被篡改 |
| ★★★★★ 深渊 | 18 | 8-12 | 没有明确任务，玩家需自行发现异常 |

## 五、多路线与结局系统

### 5.1 路线类型

每个副本至少包含 3 条路线：

- **HE（Happy Ending）：** 真相大白，正义得到伸张，关键人物得到救赎。需要高完成度的调查 + 正确的抉择。
- **TE（True Ending）：** 揭示了最深层的真相，但真相往往是残酷的。不一定是好结局，但是最"完整"的结局。
- **BE（Bad Ending）：** 调查失败、被误导、做出了错误抉择，或回合耗尽。可能有多个不同的 BE。
- **SE（Secret Ending）：** 隐藏结局。需要发现常规路线之外的隐藏线索链才能触发。

### 5.2 路线判定机制

路线不是由单一选择决定的，而是由**全程行为的累积**决定：

```
行为追踪维度：
├── 真相发现度：你找到了多少关键事实（0-100%）
├── 信任网络：哪些 NPC 信任你、哪些 NPC 警惕你
├── 道德倾向：你的行为偏向正义、中立还是利己
├── 关键节点：是否触发了特定的剧情事件
└── 时机：某些信息在特定回合之前获得才有效
```

### 5.3 结算评分

```
副本结算报告
═══════════════════════════════
副本：「沉默的铁匠铺」
结局：HE — 「迟来的清白」
═══════════════════════════════

真相还原度 ████████░░ 82%
  ✓ 发现了凶器的真正来源
  ✓ 识破了证人的谎言
  ✗ 未发现地窖中的密信（隐藏线索）

路线完成度 █████████░ 93%
  HE 线关键节点 7/8 完成

探索度 ██████░░░░ 58%
  访问地点 4/6
  深度对话 NPC 3/5

效率 █████████░ 90%
  使用回合 19/30

═══════════════════════════════
综合评级：A
回响点奖励：+350
命运余量变化：+5
═══════════════════════════════
未解锁路线：TE「无人知晓的真相」/ SE「第七个名字」

```

## 六、NPC 系统

### 6.1 NPC 分类

游戏中的 NPC 分为两大类：

**A. 回响之间常驻 NPC（元世界）**

这些 NPC 是和玩家一样被困在回响之间的灵魂，他们跨副本存在，有持续的记忆和关系发展。

| NPC | 身份 | 作用 |
|-----|------|------|
| **阿寂** | 最早的被困者之一，记忆商店老板 | 交易、情报、世界观补充。话少，但每句话都有深意。 |
| **林鹿** | 死于实验室事故的生物学研究生 | 副本攻略交流、线索分析伙伴。热心但有时过度解读。 |
| **老赵** | 死于工地事故的建筑工人 | 提供朴素但实用的建议。是回响之间的"人情世故专家"。 |
| **？？？** | 身份不明，总是戴着兜帽 | 偶尔出现，提供关于回响之间本身的隐晦信息。疑似知道回响柱的秘密。 |

这些常驻 NPC 会记住玩家的每次对话，关系会随时间发展。他们也有自己的故事线——这构成了游戏的**元叙事暗线**。

**B. 副本内 NPC**

每个副本独立的角色，副本结束后不再存在。他们是副本剧情的核心。

每个副本 NPC 拥有：
- **角色设定文档：** 性格、背景、动机、秘密、知识边界
- **关系网：** 与其他副本 NPC 的关系
- **信任度系统：** 对玩家的信任从 0 开始，影响愿意透露的信息深度
- **情绪状态：** 会根据对话内容和事件发展动态变化
- **谎言逻辑：** 如果这个 NPC 会说谎，设定中会明确谎言内容和说谎动机

### 6.2 NPC 交互机制

**信任度等级：**

```
0-20  戒备：只会说表面信息，可能敷衍或拒绝回答
20-40 中立：正常交流，会回答一般性问题
40-60 友好：开始透露个人看法和非公开信息
60-80 信任：愿意分享秘密，可能主动提供线索
80+   深信：会告诉你"不该说的话"，甚至帮你做事
```

**提升信任度的方式：**
- 表达共情和理解
- 帮助解决 NPC 的个人困扰
- 展示你已经知道的相关信息（让 NPC 觉得你值得信任）
- 多次对话积累

**降低信任度的行为：**
- 直接质问敏感话题
- 被 NPC 发现你在套话
- 向其他 NPC 泄露了该 NPC 的秘密
- 威胁或施压

## 七、元叙事：回响之间的秘密

### 7.1 表层叙事

玩家被告知的故事：你意外死亡，被回响之间拦截，完成试炼就能复活。

### 7.2 深层真相（玩家需要通过多个副本逐步发现）

回响之间并不是什么"拯救灵魂"的善意空间。

**第一层真相：** 回响柱不是在"帮助"灵魂，而是在**收割**。每次副本中灵魂经历的情感波动——恐惧、愤怒、悲伤、顿悟——都是回响柱的"养料"。副本的设计本身就是为了最大化情感产出。

**第二层真相：** 所谓的"命运余量"是虚构的概念。没有人真的能通过试炼复活。那些"通关离开"的灵魂，实际上是被回响柱彻底吸收了。

**第三层真相：** 但也不完全如此——极少数灵魂确实找到了真正的出路。线索散落在不同副本的隐藏结局中。那个神秘的兜帽人似乎知道些什么……

> 这条元叙事线不会在 MVP 中完整展开，但需要从第一个副本开始就埋下伏笔。
> 例如：副本结算时，回响柱的共鸣声中偶尔夹杂着不该存在的低语。

## 八、技术映射

### 8.1 Agent 架构对应

| 游戏概念 | 技术实现 |
|---------|---------|
| GM（叙事者） | GM Agent：负责环境描写、场景切换、行为判定、结算评分 |
| 副本 NPC | NPC Agent：每个 NPC 一个独立 agent，携带角色设定 prompt |
| 回响柱规则 | System Prompt 中的世界规则层，所有 agent 共享 |
| 信任度/情绪 | Agent 状态管理，每次对话后更新数值 |
| 线索系统 | GM Agent 维护的结构化数据，追踪玩家已发现的事实 |
| 路线判定 | GM Agent 根据行为追踪数据 + 预设条件判定 |
| 结算评分 | GM Agent 将玩家提交的推理与预设真相进行语义比对 |

### 8.2 数据结构概览

```
infinite-echoes/
├── docs/
│   ├── worldview.md          ← 你正在看的这个文件
│   └── episodes/
│       └── ep01-silent-forge/ ← 第一个副本
│           ├── setting.md     ← 副本世界设定
│           ├── npcs/          ← 每个 NPC 的角色文档
│           ├── clues.md       ← 线索链设计
│           ├── routes.md      ← 路线与结局设计
│           └── script.md      ← 关键剧情节点脚本
├── src/                       ← 游戏代码
└── README.md
```

## 九、MVP 范围定义

### 第一个副本：「沉默的铁匠铺」

- 1 个村庄地图，6 个地点
- 5 个 NPC，各有完整角色设定
- 1 条主线案件
- 3 条路线（HE / TE / BE）+ 1 条隐藏线（SE）
- 30 回合上限
- 完整的对话、探索、推理提交、结算流程

### 回响之间（简化版）

- 中央广场 + 记忆商店
- 2 个常驻 NPC（阿寂、林鹿）
- 副本入口机制
- 基础的结算和存档
